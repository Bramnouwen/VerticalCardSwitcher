## Installation

VerticalCardSwitcher is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```swift
platform :ios, '8.0'
use_frameworks!

target 'MyApp' do
pod "VerticalCardSwitcher"
end
```
